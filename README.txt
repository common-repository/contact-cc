== Description ==

**OVERVIEW**

The aim of Contact CC is to make contact forms creation as easy and hassle-free as possible. Contact CC brings **six** easily customizable, **responsive**, professional templates which will make adding a contact form to your WordPress website a breeze.

 With Contact CC you can build an elegant contact form in a couple of minutes. Make it match your website’s design, preview your changes live while editing the content in our custom visual editor and embed the form easily with a shortcode.

Contact CC is a perfect way to build your mailing lists or to keep in touch with your visitors.

== Installation ==


= Using The WordPress Dashboard =


1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'chch-contact'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Go to Contact CC Free in the left menu to manage the contacts


= Uploading in WordPress Dashboard =


1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select 'chch-contact' from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
6. Go to Contact CC Free in the left menu to manage the contacts


= Using FTP =


1. Download 'chch-contact'
2. Extract the 'chch-contact' directory to your computer
3. Upload the 'chch-contact' directory to the '/wp-content/plugins/' directory
4. Activate the plugin in the Plugin dashboard
5. Go to Contact CC Free in the left menu to manage the contacts
